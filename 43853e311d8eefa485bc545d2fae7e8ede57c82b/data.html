The Challenge

An award-winning startup provides cutting-edge IT infrastructure for data-driven companies. Its adaptable technology empowers enterprises to build their own data centers on the fly efficiently. However, its user interface was too complex to be understood by even the experienced IT technicians, which hindered user adoption.

The Solution

We completely redesigned the platform from scratch to make the product simpler and more intuitive while maintaining the core features of the product.

Design Process

The client has a tight budget and timeline. We have to move fast and adapt to changes quickly. We completed the whole project from discovery to hi-fi design delivery within 3 months.

User Workflow

We’ve conducted 8 user interviews with current and prospective users to understand their workflows. Based on the user needs and pain points, we proposed design features that will improve the overall user experience.

Conceptual Design

We created a new conceptual model, which is a design blueprint that outlines the new organizational structure of the system. We used it as a framework to list out major objects, and mark the connection between different objects. We also worked with the client to understand all those connections and come up with a business strategy and implementation plan that is the best fit based on the client’s needs.

Design Impact

We have been working closely with the directors, product managers, and engineers throughout the whole process on this highly visible project.
